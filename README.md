# 📚 宗教課選擇題測驗

根據《ethan_test.pdf》題庫生成的互動選擇題測驗網頁。

## 功能

- ✅ 選擇特定課文或全部課文
- ✅ 隨機抽取題目
- ✅ 即時顯示正確/錯誤答案
- ✅ 顯示正確答案同解釋
- ✅ 計分系統
- ✅ 手機/電腦都適用

## 課文內容

| 課數 | 主題 |
|------|------|
| 第九課 | 嫉妒傷人 (撒烏耳與達味) |
| 第十課 | 貪心招禍 (納阿曼) |
| 第十一課 | 納阿曼的僕人 |
| 第十二課 | 以斯帖 (陰謀被揭) |
| 第十三課 | 迷色輕薄人 (若瑟) |

## 使用方法

1. 選擇想測驗的課文
2. 按「開始測驗」
3. 揀選答案
4. 睇結果同解釋
